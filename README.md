### go-workout-01 ###
Go's workout 01, based on [Learn Go with Tests](https://quii.gitbook.io/learn-go-with-tests/)

#### Go commands ####
To run test
```shell
go test
```
To run test with coverage
```shell
go test -cover
```
To run test (verbose)
```shell
go test -test.v
```
To run a (particular) test
```shell
go test -run <TestName>
go test -run TestArea
```
